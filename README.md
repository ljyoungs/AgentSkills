# AgentSkills

> WorkBuddy AI 助手技能集合 — 可直接加载到 [WorkBuddy](https://www.codebuddy.cn/workbuddy) 使用的自动化技能包

---

## 技能列表

| 技能 | 说明 | 版本 |
|------|------|------|
| [douyin-traffic-psy](./douyin-traffic-psy/) | 抖音智能引流助手 — 评论扫描 + 关键词匹配 + 半自动回复 | V3.0 |

---

## 使用方式

1. Clone 本仓库
2. 进入对应技能目录
3. 按技能目录下的 `README.md` 配置
4. 在 WorkBuddy 中加载技能目录

```bash
git clone https://github.com/ljyoungs/AgentSkills.git
```
